Gemini API

模型計價（USD）

用於計算翻譯成本，單位為「每 1M tokens 的美金價格」。切換模型時會自動帶入參考價，你也可以手動修改。請至 Gemini 官方定價頁 查詢最新費率

效能（翻譯效率調校）

調整每批翻譯的大小與並行數量，影響翻譯速度與品質的平衡

配額（API 用量限制）

根據你的 Gemini API 付費層級，自動控制請求頻率，避免超出 Google 的使用量限制。你的目前層級與配額可至 Gemini API Rate Limit 查詢

LLM 參數微調

術語表一致化

長文翻譯時，先擷取全文專有名詞建立術語對照表，確保分批翻譯的名詞譯名一致。短文（1 批以內）自動跳過。

網域規則

語言偵測

開啟時，若整頁文字以繁體中文為主，按翻譯會直接跳過。關閉後仍會逐段跳過中文段落，只翻譯外語內容。Gmail 等介面為中文但信件多為英文的網站，建議關閉此選項

翻譯進度通知

匯入 / 匯出設定

匯出為 JSON 檔案備份，或從檔案匯入還原（API Key 不包含在匯出入範圍）

快捷鍵

預設 Option + S 切換目前分頁的翻譯。 可至 chrome://extensions/shortcuts 變更。

回復預設設定

將所有設定（模型、參數、計價、網域規則、系統提示等）還原為預設值。API Key 會被保留，不需要重填。翻譯快取與累計使用統計也不會受影響（請分別在 Popup 點「清除快取」與「重置統計」處理）。
